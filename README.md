# Binary sentiment classification of IMDb movie reviews using word2vec-embedding

## Dataset source
Free to download at
https://www.kaggle.com/c/word2vec-nlp-tutorial/data

## Dependencies
`torch==1.4.0` <br>
`torchtext==0.5.0` <br>
`nltk==3.5` <br>
`matplotlib` <br>
`cv2` <br>

## Experiment with 5 different recurrent approaches

Model 0. Vanilla RNN with word embedding<br>
Model 1. GRU with word embedding<br>
Model 2. LSTM with word embedding<br>
Model 3. Bidirectional LSTM with word embedding <br>
Model 4. LSTM without word embedding<br>
